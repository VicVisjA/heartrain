# 💖 Love Rain – Interactive Heart Game

A fun and romantic mini web experience where hearts fall from the sky and reveal sweet love messages when clicked. Built with pure HTML, CSS, and JavaScript, this project combines simple animations with interactive elements to create a playful and engaging UI.

---

## ✨ Features

- 💕 Animated falling hearts using CSS keyframes  
- 🎯 Click interaction to reveal random love messages  
- ✨ Smooth fade-out text animations  
- 🎨 Soft gradient background for a romantic aesthetic  
- ⚡ Lightweight and runs directly in the browser (no dependencies)

---

## 🚀 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla)** – Interactivity & DOM manipulation  

---

## 🎮 How It Works

Hearts continuously fall from the top of the screen.  
When you click on one, it "pops" and displays a random romantic message before fading away.
